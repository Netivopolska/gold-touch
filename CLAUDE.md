# Gold Touch — Claude Instructions

## Frontend Design

When writing or editing any frontend code (HTML, CSS, JavaScript, UI components), always apply the `frontend-design` skill principles:

- Ground the design in the subject's own world — avoid generic templates
- Work in two passes: first plan (colors, typography, layout, signature element), then critique before coding
- Pair typefaces deliberately; avoid the three common defaults (cream/serif/terracotta, dark/neon, newspaper/hairline)
- One bold signature element; keep everything else disciplined and restrained
- Writing in UI: active voice, specific over clever, consistent naming
